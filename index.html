What is Linked List?

Linked List는 Node들이 한줄로 연결되어 있는 방식의 자료구조이다.

Data Structure의 대상은 Memory이다.
따라서 Memory를 효율적으로 사용하는것이
Data Structure의 목적이다.

STORAGE –데이터–> MEMORY –데이터–> CPU

MEMORY의 특징은 각각 address에 접근하는데 걸리는 시간이 같다.
이를 RAM(Random Access Memory)라고 부른다.

*address란 data가 저장되는 주소.

Linked List의 구조

*Node/vertex: Element들의 모임
Node는 Data field와 Link field로 이루어져있다.

How Linked List works

15->6->23->->4->7->71

  1. 새 node 생성
  2. 새 node의 다음 node로 head르 가리킨다
  3. 새 node가 head가 되도록 head의 값을 변경한다.

What is Bound Method?

Python에서 method는 object에 붙어있다 면을 제외하고는 function과 같다.
method는 object에 불러지며 그 object에 변화를 준다.
이 method들은 bound, unbound 또는 static method가 될 수 있다.
static method는 unbound method의 한 종류이다.

Bound Methods

만약에 function이 class의 속성이고 instance에 엑세스된다. 그것들은 bound method라고 불린다.
Bound method는 ‘self’를 첫번째 argument로 가지고 있다.
이들은 class의 instance에 의존적이므로 instance method라고도 불린다.

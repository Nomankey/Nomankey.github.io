Why Error Happens

error가 발생하는 이유는 프로그램이 잘못 작동하는 것을 막기위한 파이썬의 배려이나 때때로 오류를 피하고 싶을때가 있다. 이를 위해 파이썬은 try, except를 사용해서 예외적으로 오류를 처리할 수 있게 해준다.

Error Avoid Techniques

Basic ex)

Try 블록 도중에 오류가 발생하면 except 블록이 수행된다. 하지만 try 블록에서 오류가 발생하지 않으면 except 블록은 수행되지 않는다.

발생 오류와 오류 메세지 변수까지 포함한 except 문 ex)

Try.. Finally ex)

또 try 문에는 finally 절을 사용할 수 있는데 finally 절은 try 절과는 관계없이 항상 수행된다. 보통 finally절은 사용한 리소를 close해야 할때에 많이 사용한다.

오류를 일부러 발생시키기

except문에 pass를 써서 오류를 피할 수 있지만 raise로 일부러 발생시키는 이유도 있다.

**** NotImplementedError: 파이썬 내장 오류로서, 꼭 작성해야하는 부분이 구현되지 않았을 경우 일부러 오류를 일으키기위해 사용된다.
